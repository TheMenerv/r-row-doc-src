The `PointOptions` interface represents the options for configuring a point in the game.

<br/>

## Definition

```ts
interface PointOptions {
  color?: string;
  weight?: number;
}
```

<br/>

## Properties

### color

- Type: `string` (optional)

The color of the point.

### weight

- Type: `number` (optional)

The weight (size) of the point.
