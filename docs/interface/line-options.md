The `LineOptions` interface represents the options for configuring a line in the game.

<br/>

## Definition

```ts
interface LineOptions {
  color?: string;
  weight?: number;
}
```

<br/>

## Properties

### color

- Type: `string` (optional)

The color of the line.

### weight

- Type: `number` (optional)

The weight (thickness) of the line.
