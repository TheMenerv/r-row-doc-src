The `NineSliceMode` enum represents the different modes of a nine-slice object in the game.

<br/>

## Definition

```ts
enum NineSliceMode {
  Repeat = 'repeat',
  Stretch = 'stretch',
}
```

<br/>

## Enum Values

- `Repeat`: The nine-slice object will repeat the image.
- `Stretch`: The nine-slice object will stretch the image.
